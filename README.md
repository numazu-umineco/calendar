# calendar

イベント情報を管理していくカレンダーアプリ

## Development

### Getting started

```
docker compose up -d
```

### ディレクトリ構成

```
├── admin   ....... イベント登録管理者用フロントエンド
├── api     ....... API サーバー (Rails)
└── website ....... 一般公開用ウェブページ
```
